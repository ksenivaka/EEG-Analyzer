## Предобработка сигналов электроэнцефалограмм
### Трек Большие данные ИТ Академии Samsung
Авторы проекта: Луговских А.В., Сазонова К.А.

Руководитель: Рудыко М.О.

Разработано в Курском государственном университете.
***
Веб-приложение включает следующие функции:
* открытие файлов .vhdr, .vmrk, .eeg
* движение по временной шкале и выбор каналов для просмотра
* понижение частоты дискретизации
* фильтрация
* метод независимых компонент (ICA)
* проекция сигнального пространства (SSP)
***
Исследуемый датасет: 

https://www.kaggle.com/datasets/exten113/cssv-erp/data
